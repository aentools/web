We will present here an interdisciplinary integrated research report describing computational modeling techniques and analytical programming in a Unix environment (routines to automate data extraction and processing with Shell, Python, and Java scripts) to provide an overview of recent advances in big data and Machine Learning (ML) concepts applied in chemistry.

In addition, we will demonstrate the applications of these techniques for simulating energy storage devices known as Electrical Double-Layer Supercapacitors (EDLS) using simulation packages (Molecular Dynamics and Density Functional Theory).



ML problems can be divided into three broad classes:

  • Supervised

  • Unsupervised

  • Reinforcement

Supervised learning is used when you have previous data with results (labels in ML terminology) and you want to predict the results for the future. On the other hand, unsupervised learning mainly deals with unlabeled data, allowing the model to work on its own to discover patterns and information that have not been previously detected.
